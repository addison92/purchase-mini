#### git提交

##### 查看本地修改记录

git status


##### 添加需要推送到 git 仓库的文件/文件夹 到本地暂存区

git add .                                            添加所有改动文件
git add 文件夹/文件路径  	        添加指定文件

	
##### 给刚刚添加的文件 添加修改注释

git commit -m "修改活动申请"


##### 推送到 远端仓库

git push
